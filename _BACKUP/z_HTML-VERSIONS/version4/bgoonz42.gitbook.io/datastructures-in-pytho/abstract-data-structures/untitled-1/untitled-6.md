# <span class="text-4505230f--DisplayH900-bfb998fa--textContentFamily-49a318e1">Recursion</span>

<span class="text-4505230f--UIH300-2063425d--textUIFamily-5ebd8e40--text-8ee2c8b2"></span>

<span class="text-4505230f--UIH300-2063425d--textUIFamily-5ebd8e40--text-8ee2c8b2"></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="4278bf3bfa6f4ed0a953100dc80ca52a"><span data-offset-key="4278bf3bfa6f4ed0a953100dc80ca52a:0">When a function call itself is knows as recursion. Recursion works like loop but sometimes it makes more sense to use recursion than loop. You can convert any loop to recursion.</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="7cc039c23c404442994478ddfcb2735f"><span data-offset-key="7cc039c23c404442994478ddfcb2735f:0">Here is how recursion works. A recursive function calls itself. As you you'd imagine such a process would repeat indefinitely if not stopped by some condition. This condition is known as base condition. A base condition is must in every recursive programs otherwise it will continue to execute forever like an infinite loop.</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="5a132354ad184beebf42b2de172b9c98"><span data-offset-key="5a132354ad184beebf42b2de172b9c98:0">Overview of how recursive function works:</span></span></span>

1.  <span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="573cd153b4b0486297afa9f15ccd9009"><span data-offset-key="573cd153b4b0486297afa9f15ccd9009:0">Recursive function is called by some external code.</span></span></span>

2.  <span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="90953a9c8e844161aa57b6681f7c8384"><span data-offset-key="90953a9c8e844161aa57b6681f7c8384:0">If the base condition is met then the program do something meaningful and exits.</span></span></span>

3.  <span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="7023642ebb044ed9bac15b53981b2fc7"><span data-offset-key="7023642ebb044ed9bac15b53981b2fc7:0">Otherwise, function does some required processing and then call itself to continue recursion.
