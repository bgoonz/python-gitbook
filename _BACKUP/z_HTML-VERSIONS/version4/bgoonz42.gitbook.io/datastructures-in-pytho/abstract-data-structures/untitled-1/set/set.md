Sets manipulated by algorithms can grow, shrink, or otherwise change over time, we call such sets </span><span data-offset-key="5694ca5c1eee4e67aeea53a8f3534857:1">**dynamic**</span><span data-offset-key="5694ca5c1eee4e67aeea53a8f3534857:2">.
