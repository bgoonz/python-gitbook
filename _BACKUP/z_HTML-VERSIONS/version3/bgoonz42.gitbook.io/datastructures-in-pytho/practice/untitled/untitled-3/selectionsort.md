# <span class="text-4505230f--DisplayH900-bfb998fa--textContentFamily-49a318e1">SelectionSort</span>

<span class="text-4505230f--UIH300-2063425d--textUIFamily-5ebd8e40--text-8ee2c8b2"></span>

<span class="text-4505230f--UIH300-2063425d--textUIFamily-5ebd8e40--text-8ee2c8b2"></span>

<span class="text-4505230f--HeadingH600-23f228db--textContentFamily-49a318e1"><span data-key="df7b79deeba9423d82bc5c2bcf40a2e4"><span data-offset-key="df7b79deeba9423d82bc5c2bcf40a2e4:0">What is Selection Sort?</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="3b46fabad12d4ec2b30627c1205c8fa7"><span data-offset-key="3b46fabad12d4ec2b30627c1205c8fa7:0">In computer science, it is a </span></span><a href="https://en.wikipedia.org/wiki/Selection_sort" class="link-a079aa82--primary-53a25e66--link-faf6c434"><span data-key="ba3f3a5a56ca46069ba04d40ba823af2"><span data-offset-key="ba3f3a5a56ca46069ba04d40ba823af2:0">sorting algorithm</span></span></a><span data-key="bede6741e106440490e88b6a695f6773"><span data-offset-key="bede6741e106440490e88b6a695f6773:0">, specifically an in-place comparison sort.</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="a2dbe0a4984e407399256db75e5dabf4"><span data-offset-key="a2dbe0a4984e407399256db75e5dabf4:0">It has O(n2) time complexity, making it inefficient on large lists, and generally performs worse than the similar insertion sort.</span></span></span>

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="29e903bc297844d7a0f0a6ebe0510783"><span data-offset-key="29e903bc297844d7a0f0a6ebe0510783:0">Selection sorting is noted for its simplicity, and it has performance advantages over more complicated algorithms in certain situations, particularly where auxiliary memory is limited.</span></span></span>

###

<span class="text-4505230f--TextH400-3033861f--textContentFamily-49a318e1"><span data-key="d6ca9bb079514f4f83dafb5a93b545ee"><span data-offset-key="d6ca9bb079514f4f83dafb5a93b545ee:0">So for this algorithm, we are going to use two for loops, one for traversing through each element from index 0 to n-1.</span></span></span>

    def selectionSort(List):    for i in range(len(List) - 1):        minimum = i        for j in range( i + 1, len(List)):            if(List[j] < List[minimum]):                minimum = j        if(minimum != i):            List[i], List[minimum] = List[minimum], List[i]    return List
